# numeral-cli

command line interface to the awesome numeral.js

### installation

    npm i -g numeral-cli


### usage

    ./index.js -h
    Usage: node ./index.js -i [inputfile] -o [outputfile] -f [format] -s [separator]

    Options:
      -f, --format     [default: ""]
      -i, --input      [default: "stdin"]
      -o, --output     [default: "stdout"]
      -s, --separator  [default: "\n"]
